# lib-argparse

lib-argparse is a Rust library for parsing arguments.

## Usage

Place the following into your `Cargo.toml`:

```toml
[dependencies]
argparse.git = "https://github.com/shipsimfan/lib-argparse"
```

See the documentation on in [lib.rs](./src/lib.rs).

## Contributing
